# BytesTutor

BytesTutor is a tutoring application with a native iOS client and a robust Go backend.

## Project Structure

- `/backend` - Go Backend REST API built with Gin, structured in Clean Architecture.

## Tech Stack & Architecture

- **Backend**: Go (Golang), Gin Web Framework, PostgreSQL Database (Custom schema `tutora_app`), Versioned migrations with `golang-migrate`.
- **Architecture**: Clean Architecture (Domain, Repository, Usecase, Delivery).
